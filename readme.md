# Projeto JavaScript - Curso SENAI

Este repositório contém o início do desenvolvimento de um projeto prático realizado durante o curso de JavaScript do SENAI. O objetivo é aplicar conceitos fundamentais de programação, versionamento e organização de software.

# Tecnologias utilizadas

- JavaScript (em aprendizado)
- Git (controle de versão)
- GitHub (armazenamento do repositório)

```bash
/projeto-js-senai  
│-- src/  
│-- README.md  
│-- .gitignore  

```

# Versionamento com Git

Este projeto utiliza o Git para controle de versão, permitindo o registro de alterações no código, a criação de histórico de desenvolvimento e a possibilidade de trabalho colaborativo.

Comandos básicos utilizados:

```bash

git init  
git add .  
git commit -m "início do projeto"  
git branch -M main  
git remote add origin https://github.com/brunohl03/Javascript.git
git push -u origin main  

```

# Repositório no GitHub

O código está hospedado no GitHub para facilitar o acesso, compartilhamento e versionamento do projeto. (https://github.com/brunohl03/Javascript.git)

# Análise de Requisitos

## Requisitos Funcionais (RF)

- O sistema deve executar funcionalidades básicas em JavaScript  
- O usuário deve conseguir interagir com o sistema  
- O projeto deve evoluir conforme os aprendizados do curso  

## Requisitos Não Funcionais (RNF)

- O código deve ser organizado e legível  
- O projeto deve utilizar versionamento com Git  
- O sistema deve ser de fácil manutenção
